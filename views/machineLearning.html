What is Machine Learning?

In machine learning, computers use algorithms to learn from stored data to make prdeictions. However, this data must be input by humans.

Example

What if we wanted a computer to be able to recognize what a dog is? First we ask ourselves how it is a human is able to tell what a dog is.
  • Shape
  • Size
  • Sound
  • Smell
A person has to input all of the features that define what a dog is appropriate to a computer. With that information, when the computer sees a dog it should now be able to tell what it is by matching its stored data with the current visual and audio input it recieves while looking at the dog.
